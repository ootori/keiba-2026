# 新馬戦における大外枠の回収率分析レポート

## 概要

「新馬戦では大外枠が有利（ゲート入りに慣れていない馬が多く、最後にゲートインする大外の馬が落ち着いてスタートできる）」という競馬の格言が実データで裏付けられるかを検証した。
JRA中央10場の2015-2025年（11年分、約23.5万出走、16,515レース）を対象に、大外枠（各レースの最大馬番）の勝率・複勝率・回収率を、それ以外の枠順と比較した。

## 分析方法

### データソース
- テーブル: `n_uma_race`（出走情報）、`n_race`（レース条件）、`n_harai`（払戻データ）
- 対象: JRA中央10場、確定成績（datakubun='7'）、出走取消なし（ijyocd='0'）
- 新馬戦の判定: `jyokencd5 LIKE '7%'`（2歳新馬・3歳新馬）

### 大外枠の判定ロジック
- 各レースにおいて最大馬番の馬を「大外枠」と判定
- 出走取消馬（ijyocd≠'0'）は除外済みのため、実際に出走した中での最大馬番

### カテゴリ分類
| カテゴリ | 定義 |
|---------|------|
| 大外枠 | レース内で最大馬番の馬 |
| 外枠(大外以外) | 馬番が頭数の上位25%（大外を除く） |
| 中枠 | 馬番が頭数の中央50% |
| 内枠 | 馬番が頭数の下位25% |

### 回収率の計算
- 各カテゴリに該当する全出走馬の単勝/複勝を100円ずつ購入した場合の回収率
- `回収率(%) = 払戻金合計 / (100円 × 購入数)`

### 新馬戦の頭数分布
- 平均: 14.2頭、中央値: 15頭、最小: 5頭、最大: 18頭

## 結論

### 「新馬戦では大外枠が有利」は成績面では部分的に裏付けられるが、回収率では裏付けられない

| 指標 | 全期間差分（大外枠 − その他） |
|------|--------------------------|
| 勝率 | **+0.67pt** (7.66% vs 6.99%) |
| 複勝率 | **+1.11pt** (22.12% vs 21.01%) |
| 単勝回収率 | **-2.2pt** (69.5% vs 71.7%) |
| 複勝回収率 | **-0.6pt** (71.0% vs 71.6%) |

### 成績面の分析

- **勝率:** 大外枠は+0.67ptとその他を上回る。11年中8年でプラス、2023-2025年は+0.69〜+1.91ptと近年ほど差が拡大している
- **複勝率:** 大外枠は+1.11ptとその他を上回る。11年中9年でプラス。こちらも近年ほど差が大きく、2023年+2.50pt、2024年+3.03pt、2025年+3.16ptと年々拡大傾向にある
- **格言の根拠は実データで確認できる:** ゲート入りの順序による落ち着きの差が実際の走力発揮に影響している可能性がある

### 回収率面の分析

- **単勝回収率:** 大外枠は-2.2ptとその他を下回る。11年中プラスは4年のみ（2015, 2017, 2020, 2023）。振れ幅が非常に大きく（-20.9pt〜+30.8pt）、安定しない
- **複勝回収率:** 大外枠は-0.6ptとほぼフラットだが、わずかに下回る。11年中プラスは4年のみ
- **オッズへの織り込み:** 大外枠の成績面の有利さはすでに市場に認知されており、オッズに適正に（あるいはやや過剰に）織り込まれている。そのため、成績は上回るが回収率では優位にならない

### 条件別の注目ポイント

- **頭数帯:** 多頭数（15頭以上）では大外枠の勝率・複勝率がその他とほぼ同等。少頭数（~10頭）では大外枠の勝率が+1.02pt高いが回収率は-6.4pt。**頭数によらず回収率での優位性はない**
- **芝/ダート:** ダートでは大外枠の勝率が+1.19ptと芝より差が大きい。ただし回収率はダートでも-3.1ptとマイナス
- **馬番グループ:** 4-6番が勝率7.43%・回収率74.8%で最も効率が良い。13番以降は勝率6.27%・回収率68.2%と最低。大外枠の好成績は「最後にゲート入りする」効果であり、外枠全般の有利さではない

### 近年のトレンド（2023-2025年）

- 複勝率での大外枠の優位性が近年拡大している（+2.50pt→+3.03pt→+3.16pt）
- これがトレンドの変化なのかサンプルの偏りかは、今後のデータ蓄積で判断する必要がある
- ただし2024年は複勝率が+3.03ptでも複勝回収率は-8.3ptであり、成績の優位がオッズに反映されている構造は変わらない

### 総合評価

**格言「新馬戦では大外枠は買い」は、成績面では確かに裏付けられる（勝率+0.67pt、複勝率+1.11pt）。しかし、この有利さは市場で認知されオッズに織り込まれているため、回収率では優位にならない（単勝-2.2pt、複勝-0.6pt）。「大外枠だから買い」という戦略は利益に結びつかない。**
