# 初ダート回収率分析レポート

## 概要

「初ダートの馬は買い」という仮説が実データで裏付けられるかを検証した。
JRA中央10場の2015-2025年（11年分、約50万出走）を対象に、初めてダートを走ったレースの勝率・複勝率・回収率を、それ以外のケースと比較した。

## 分析方法

### データソース
- テーブル: `n_uma_race`（出走情報）、`n_race`（レース条件）、`n_harai`（払戻データ）
- 対象: JRA中央10場、確定成績（datakubun='7'）、出走取消なし（ijyocd='0'）
- 障害レースは除外（芝・ダートのみを対象）

### 初ダートの判定ロジック
- 馬ごとに2001年以降の全出走履歴を時系列順にソート
- トラックコード（trackcd）で芝（10-22）/ダート（23-29）を判定
- 当該レースがダートかつ、それ以前の全レースでダートを走っていないものを「初ダート」と判定

### カテゴリ分類
| カテゴリ | 定義 |
|---------|------|
| 初ダート | 当該レースが生涯初のダート出走 |
| 2回目以降ダート | 過去にもダートを走ったことがある馬のダートレース |
| 芝のみ | 芝レースの出走（ダート未経験含む） |

### 回収率の計算
- 各カテゴリに該当する全出走馬の単勝/複勝を100円ずつ購入した場合の回収率
- `回収率(%) = 払戻金合計 / (100円 × 購入数)`

## 結論

### 「初ダートの馬は買い」は一概には言えない

| 指標 | 全期間差分（初ダート − 芝のみ） |
|------|------------------------------|
| 勝率 | **-1.44pt** (5.90% vs 7.34%) |
| 複勝率 | **-5.24pt** (16.76% vs 22.00%) |
| 単勝回収率 | **+2.3pt** (74.2% vs 71.9%) |
| 複勝回収率 | **-4.6pt** (68.0% vs 72.6%) |

- **成績面:** 初ダート馬は勝率・複勝率ともに芝のみレースを大きく下回る。芝で通用しなかった馬がダートに転向するケースが多く、能力的に劣る集団が含まれることが主因と考えられる
- **単勝回収率:** 全期間で芝のみを+2.3pt上回る。初ダート馬は実績がないため人気になりにくく、オッズが甘くなる傾向がある。年度別では11年中7年で芝のみを上回っており、やや安定した傾向
- **複勝回収率:** 全期間で芝のみを-4.6pt下回る。複勝では穴をあけるほどの爆発力がなく、回収率が振るわない。11年中10年でマイナスと極めて安定したマイナス傾向
- **年度安定性:** 単勝では不安定（-24.1pt〜+16.8ptの振れ幅）だが、年間3,500〜4,000件程度のサンプルサイズでは分散が大きくなるのは自然。複勝ではほぼ一貫してマイナス

### まとめ

初ダート馬は単勝回収率でわずかに優位だが、複勝率・複勝回収率では明確に劣後する。「初ダートの馬は単勝で穴を狙う」という限定的な戦略には一定の根拠があるものの、安定的に儲かる手法とは言い難い。2回目以降ダート馬が全指標で最も安定しており、ダート適性が確認された馬の方が信頼できる。
