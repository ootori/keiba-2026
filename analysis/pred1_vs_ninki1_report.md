## 4. 分析まとめ

### モデル再学習による変化

| 観点 | 影響 |
|------|------|
| 単勝 | 194% → 95%に大幅低下。1着予測の精度がこのゾーンで落ちた |
| 馬単A（→予想2位） | 60.7% → 0.0%。2着予測は改善されず、むしろ悪化 |
| **馬単B（→人気1位）** | **79.7% → 130.7%に改善。唯一の黒字戦略に** |

### 馬単B（予想1位→人気1位）の特徴

- 2回の的中で+16,070円を回収し、123レース分の投資を上回った
- 特に**オッズ20~50倍帯で回収率462.8%** と大きな配当を獲得
- 人気1位は2着に来やすいため、相手馬の選定として合理性がある
- ただし的中率1.6%（2/123）とサンプルが極めて少なく、安定性は不明

### 馬単A（予想1位→予想2位）の課題

- 前回も今回も的中率が極めて低い（0.7% → 0.0%）
- モデルの2着予測精度は馬単を成立させるレベルに達していない
- 2着の予測精度向上は引き続き課題

## 5. 結論と推奨戦略

### 現時点の推奨

1. **単勝買い** が最もシンプルかつ安定的。ただし再学習で回収率が変動するリスクあり
2. **馬単B（→人気1位）** は今回の再学習で唯一の黒字戦略。人気1位を2着相手に固定する合理性は高いが、的中率の低さから収支のブレが大きい

### 馬単Bが有効な条件（仮説）

- **オッズ20~50倍帯** で特に効果的（人気薄×人気1位の馬単は高配当になりやすい）
- **人気5位前後** の予想1位馬で回収率が高い傾向

### 今後の検討事項

1. **馬単Bのサンプル拡大:** 複数年（2023-2025等）で同様の傾向が出るか検証
2. **馬単の多点買い:** 予想1位→人気1~3位の3点買いでカバー率を上げる戦略
3. **ワイド・馬連での検証:** 馬単より的中幅が広い券種との比較
4. **モデル安定性:** 再学習による回収率変動を抑える工夫（アンサンブル等）
