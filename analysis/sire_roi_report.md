# 種牡馬別回収率分析レポート

**分析日:** 2026-02-21
**対象期間:** 2019〜2024年（6年間）
**対象:** JRA中央10場、確定成績（DataKubun='7'）、出走取消除外
**データ件数:** 213,399出走、639種牡馬
**全体ベースライン単勝回収率:** 72.9%
**分析スクリプト:** `analysis/sire_roi_analysis.py`

---

## 1. 分析の目的

種牡馬ごとに単勝回収率を算出し、サーフェス（洋芝/芝/ダート）、競馬場、距離帯の各軸で回収率に有意な差があるかを検証する。回収率に偏りがある場合、市場（ファン）が種牡馬の条件別適性をオッズに十分反映できていない（＝オッズに歪みがある）ことを意味する。

## 2. サーフェス別（洋芝/芝/ダート）

### 分類定義

| サーフェス | 条件 |
|-----------|------|
| 洋芝 | 札幌（jyocd=01）・函館（jyocd=02）の芝コース（trackcd < 23） |
| 芝 | 上記以外の芝コース（trackcd < 23） |

- 洋芝は標準偏差42.4%と最大で、種牡馬間のばらつきが大きい。開催が札幌・函館に限られサンプル数が少ないことが原因

## 7. 総括

### 歪みの存在は明確

1. **芝/ダート間の乖離が最大。** 同一種牡馬で芝/ダートの回収率差が100ポイントを超えるケースが複数存在し、市場が種牡馬のサーフェス適性をオッズに十分反映できていない。

2. **洋芝（札幌・函館）のばらつきが最大。** 標準偏差42.4%、回収率>100%が25.9%と3サーフェス中最高。サンプルの少なさにより歪みが大きくなっている。

3. **長距離ほど種牡馬間の格差が大きい。** 標準偏差はスプリント37.5%に対し長距離49.1%。長距離レースの血統適性はオッズに反映されにくい。

4. **ローカル場で穴種牡馬の回収率が高い。** 福島・新潟・小倉で300%超の回収率が散見され、主要場に比べ購買者の種牡馬分析が薄い可能性。

5. **サーフェス×距離のクロス効果も大きい。** ドゥラメンテのダートスプリント131.6% vs ダート中距離18.2%のように、同一サーフェス内でも距離条件で100ポイント超の差がある。
