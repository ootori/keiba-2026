# 初ブリンカー回収率分析レポート

## 概要

「初ブリンカーの馬は買い」という競馬の格言が実データで裏付けられるかを検証した。
JRA中央10場の2015-2025年（11年分、約52万出走）を対象に、初めてブリンカーを装着したレースの勝率・複勝率・回収率を、それ以外のケースと比較した。

## 分析方法

### データソース
- テーブル: `n_uma_race`（出走情報）、`n_harai`（払戻データ）
- 対象: JRA中央10場、確定成績（datakubun='7'）、出走取消なし（ijyocd='0'）

### 初ブリンカーの判定ロジック
- 馬ごとに2001年以降の全出走履歴を時系列順にソート
- 当該レースで `blinker='1'`（使用）かつ、それ以前の全レースで `blinker='0'`（不使用）だったものを「初ブリンカー」と判定
- 2001年以降を参照範囲としたのは、馬齢表記が満年齢に変更された年であり、プロジェクトで使用可能なデータの起点であるため

### カテゴリ分類
| カテゴリ | 定義 |
|---------|------|
| 初ブリンカー | 当該レースが生涯初のブリンカー装着 |
| 2回目以降ブリンカー | 過去にもブリンカーを装着したことがある馬のブリンカー装着レース |
| ブリンカーなし | ブリンカー未装着のレース |

### 回収率の計算
- 各カテゴリに該当する全出走馬の単勝/複勝を100円ずつ購入した場合の回収率
- `回収率(%) = 払戻金合計 / (100円 × 購入数)`

## 結論

### 「初ブリンカーの馬は買い」は裏付けられない

| 指標 | 全期間差分（初ブリンカー − ブリンカーなし） |
|------|----------------------------------------|
| 勝率 | **-2.40pt** (4.94% vs 7.34%) |
| 複勝率 | **-6.43pt** (15.46% vs 21.89%) |
| 単勝回収率 | **-4.3pt** (67.9% vs 72.2%) |
| 複勝回収率 | **-5.9pt** (67.4% vs 73.3%) |

- **成績面:** 初ブリンカー馬は勝率・複勝率ともにブリンカーなしを大きく下回る。ブリンカーを着ける馬は気性難であり、能力的に劣る集団が多いことが主因と考えられる
- **回収率面:** 単勝・複勝ともにブリンカーなしを下回る。「初ブリンカーで激走」への期待がオッズに織り込まれ、実力以上に買われている（過大評価）ことを示す
- **年度安定性:** 単勝回収率で初ブリンカーが上回ったのは11年中4年のみ（2017, 2020, 2021, 2024）。複勝回収率では11年中10年でマイナス。傾向は安定している
- **2020年の例外:** 初ブリンカーが唯一プラス回収（104.3%）だった年。年間1,077件のサンプルにおける偶然の偏りと考えられる

### 副次的な発見: 2回目以降ブリンカーの優位性

| 指標 | 2回目以降 − ブリンカーなし |
|------|------------------------|
| 単勝回収率 | **+6.6pt** (78.8% vs 72.2%) |
| 複勝回収率 | **+2.2pt** (75.5% vs 73.3%) |

2回目以降のブリンカー装着馬は回収率がブリンカーなしを上回っている。ブリンカーに慣れて安定走する馬は市場で過小評価されている可能性がある。
