# 初芝回収率分析レポート

## 概要

「初芝の馬は買い」という仮説が実データで裏付けられるかを検証した。
JRA中央10場の2015-2025年（11年分、約50万出走）を対象に、初めて芝を走ったレースの勝率・複勝率・回収率を、それ以外のケースと比較した。

## 分析方法

### データソース
- テーブル: `n_uma_race`（出走情報）、`n_race`（レース条件）、`n_harai`（払戻データ）
- 対象: JRA中央10場、確定成績（datakubun='7'）、出走取消なし（ijyocd='0'）
- 障害レースは除外（芝・ダートのみを対象）

### 初芝の判定ロジック
- 馬ごとに2001年以降の全出走履歴を時系列順にソート
- トラックコード（trackcd）で芝（10-22）/ダート（23-29）を判定
- 当該レースが芝かつ、それ以前の全レースで芝を走っていないものを「初芝」と判定

### カテゴリ分類
| カテゴリ | 定義 |
|---------|------|
| 初芝 | 当該レースが生涯初の芝出走 |
| 2回目以降芝 | 過去にも芝を走ったことがある馬の芝レース |
| ダートのみ | ダートレースの出走（芝未経験含む） |

### 回収率の計算
- 各カテゴリに該当する全出走馬の単勝/複勝を100円ずつ購入した場合の回収率
- `回収率(%) = 払戻金合計 / (100円 × 購入数)`

## 結論

### 「初芝の馬は買い」は裏付けられない

| 指標 | 全期間差分（初芝 − ダートのみ） |
|------|-------------------------------|
| 勝率 | **-0.72pt** (6.29% vs 7.01%) |
| 複勝率 | **-2.15pt** (18.86% vs 21.01%) |
| 単勝回収率 | **-3.6pt** (69.4% vs 73.0%) |
| 複勝回収率 | **-6.1pt** (67.4% vs 73.5%) |

- **成績面:** 初芝馬は勝率・複勝率ともにダートのみレースを下回る。ダートで実績を積んだ馬が芝に挑戦するケースが多いが、芝適性がない馬も多く含まれることが主因
- **単勝回収率:** 全期間でダートのみを-3.6pt下回る。年度別では11年中4年しかプラスにならず、不安定。初ダートと異なり、穴妙味も見られない
- **複勝回収率:** 全期間で-6.1ptと大幅にマイナス。11年中10年でマイナスであり、極めて安定した低回収率傾向
- **初ダートとの比較:** 初ダート（単勝+2.3pt）と比べて、初芝は全指標で劣位。ダート→芝の転向は芝→ダートの転向より成功率が低い

### 初ダートと初芝の比較

| 指標 | 初ダート vs 芝のみ | 初芝 vs ダートのみ |
|------|-------------------|-------------------|
| 勝率差 | -1.44pt | -0.72pt |
| 複勝率差 | -5.24pt | -2.15pt |
| 単勝回収率差 | **+2.3pt** | **-3.6pt** |
| 複勝回収率差 | -4.6pt | -6.1pt |

初ダートは成績面の劣後が大きいが、人気薄で出走するため単勝回収率はわずかにプラス。一方、初芝は成績面の劣後は小さいが、それでも回収率は全面的にマイナスとなる。

### まとめ

初芝馬は勝率・複勝率・回収率のいずれにおいてもダートのみレースを下回る。「初芝の馬は買い」という仮説はデータで裏付けられない。芝適性は血統的な要素が大きく、ダートで走っていた馬が突然芝で好走する確率は低い。2回目以降芝馬が全指標で最も高く、芝の実績が確認された馬の方が信頼できる。
