### チューニングの効果

パラメータファインチューニング（第4版→第5版）は:

- **不一致全体の回収率は微増**（84.4% → 85.2%、+0.8pt）
- **一致率がさらに上昇**（56% → 58%）。モデルが市場に近づく傾向は継続
- **10.1~20.0倍帯で単勝108.7%・馬単A 105.5%の黒字を達成** — 全5版で初めてこのオッズ帯で馬単Aが黒字化
- ただし**高オッズ全体（10倍以上）の単勝は79.8%に低下**。20倍以上帯が全滅したため
- **人気6位で回収率129.7%（37件）** — 少件数ながら安定的な黒字帯として浮上

### 馬単Aの復活について

- 第2版～第4版まで全滅だった馬単A（予想1位→予想2位）が、第5版で**1的中（8,440円払戻）**を記録
- 10.1~20.0倍帯に限ると**回収率105.5%**で黒字
- ただしたった**1的中**に依存しており、統計的信頼性は低い
- 人気6位×馬単Aで**324.6%**という高回収率は注目に値するが、これも1的中×高配当の結果

### 注目ゾーン

- **不一致×オッズ10.1~20.0倍: 回収率108.7%（80件）** — 第5版で最も有望な黒字帯。第4版の85.4%から大幅改善
- **不一致×人気2位: 回収率90.2%（582件）** — ボリュームゾーンで安定的な高水準
- **不一致×人気6位: 回収率129.7%（37件）** — 穴馬ゾーンとして有望
- **不一致×オッズ2.1~3.0倍: 回収率93.1%（88件）** — 低オッズ帯でも堅実

## 6. 結論と推奨戦略

### 現時点の評価

1. **不一致パターンは全5版を通じて一致パターンより回収率が高い**（第5版: 85.2% vs 77.8%）。市場と異なる評価に妙味がある基本構造は全モデルで一貫
2. **10.1~20.0倍帯が最有望ゾーンに浮上。** 単勝108.7%・馬単A 105.5%と、チューニングにより黒字化を達成した唯一のオッズ帯
3. **20倍以上の高オッズ帯は全5版で安定しない。** 件数不足と的中の偶然性が大きく、戦略の柱にはできない
4. **馬単Bは全5版を通じて第2版の130.7%を除き不振。** 「穴馬→人気1位」の着順は予測困難
5. **第1版（223特徴量）が不一致全体の回収率では依然最高。** 特徴量追加・チューニングで一致率が上がり、エッジが縮小する傾向

### 推奨

| 用途 | 推奨 |
|------|------|
| 穴馬単勝戦略（全オッズ帯） | 第1版モデル（223特徴量, bms+rating）が最有利 |
| **10~20倍帯限定の単勝** | **第5版が最有力（108.7%、80件）** |
| **10~20倍帯限定の馬単A** | **第5版で初の黒字化（105.5%）だが要継続検証** |
| 馬単B戦略 | 全版で非推奨 |
| 全体的な予測精度 | 第5版（チューニング済み）が最善の可能性。NDCG等での比較が必要 |

### 今後の検討事項

1. **第1版モデルの復元・保存:** 最高回収率を出した第1版モデルを別名で保存し、戦略用に維持
2. **10~20倍帯戦略の継続検証:** 第5版で黒字化した10.1~20.0倍帯の成績を過去年度でも検証（過学習でないか確認）
3. **ワイド・馬連の検証:** 馬単より的中幅の広い券種で2着予測を活かせるか
4. **人気6位ゾーンの深掘り:** 第5版で129.7%（37件）を記録した人気6位の条件をさらに絞り込み
5. **モデル使い分け戦略:** 全体精度（第5版）× 穴馬戦略（第1版）のアンサンブルやゾーン別使い分け
6. **特徴量数と回収率の関係:** 223→236で一致率が上がりエッジが減る問題の解明。特徴量を減らして再チューニングした場合の挙動確認
