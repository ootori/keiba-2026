# 馬主による意図的着順調整の統計的検証

## 調査の背景

「馬主によっては昇格すると賞金が稼げなくなることから、1着を取れるケースでも2着・3着に収めるケースがある」という噂の真偽を、JRA-VANデータ11年分（2015-2025年、約51万出走）を用いて統計的に検証した。

### JRAの収得賞金制度

条件戦（1勝/2勝/3勝クラス）では**1着のみ**収得賞金が加算される。2着・3着では加算されない。

| クラス | 昇格閾値 | 1着時の加算額 |
|--------|----------|-------------|
| 未勝利 → 1勝 | - | 400万円 |
| 1勝 → 2勝 | 501万円以上 | 500万円 |
| 2勝 → 3勝 | 1,001万円以上 | 600万円 |
| 3勝 → OP | 1,601万円以上 | 900万円 |

2019年夏に降級制度が廃止され、一度上がったクラスから戻れなくなった。このため、仮に着順調整があるとすれば2020年以降にインセンティブが強まるはずである。

---

## 分析結果の要約

**結論: 「馬主が意図的に着順を抑えている」という仮説を支持する統計的証拠は見つからなかった。**

---

## 分析内容と結果の詳細

### Step 1: 基礎統計

条件戦の1番人気勝率はクラスが上がるにつれて低下する傾向がある（1勝31.8% → 3勝29.4%）。ただしOP・重賞も29.7-29.8%であり、上位クラスほど実力が拮抗するため勝率が低下する自然な傾向として説明可能。

| クラス | N | 1番人気勝率 | 2-3着率 | 4着以下率 |
|--------|-------|-----------|---------|----------|
| 新馬 | 16,444 | 34.8% | 33.4% | 31.9% |
| 1勝 | 10,546 | 31.8% | 30.7% | 37.5% |
| 2勝 | 4,855 | 33.1% | 30.0% | 37.0% |
| 3勝 | 2,113 | 29.4% | 29.1% | 41.5% |
| OP | 1,677 | 29.8% | 29.9% | 40.4% |
| 重賞 | 1,486 | 29.7% | 30.0% | 40.4% |

2019年降級廃止の前後で、条件戦の1番人気勝率はむしろ**上昇**している（30.2% → 33.3%、p=0.00001）。仮説が正しければ廃止後に勝率が低下するはずだが、逆方向の結果。

### Step 2: 昇格閾値効果（コア分析）

「あと1勝で昇格する馬」の勝率が低いかを検証した。

- **層別Z検定**（3クラス×3人気帯、計9通り）: Bonferroni補正後に有意なのは1つだけ。しかもそれは仮説と**逆方向**（1勝クラス1番人気で昇格リスクあり群のほうが勝率+5.0ポイント高い）
- **ロジスティック回帰**（オッズ・頭数・トラック等の交絡を制御）: `promotion_risk`の係数は-0.043（OR=0.96）だが**p=0.23で有意でない**
- **差分の差分（DID）分析**（2019年降級廃止の前後比較）: 交互作用項p=0.56で**有意でない**。降級廃止は昇格リスク効果を拡大していない

### Step 3: 馬主別分析

条件戦で1-3番人気に支持された出走を馬主別に集計し、二項検定（BH-FDR補正 q<0.05）で異常値を検出した。

**N>=50（大規模馬主）: 202名**
- 勝率が有意に低い馬主: **0名**
- 2-3着率が有意に高い馬主: **0名**

**N=20-49（小規模馬主）: 217名**
- 勝率が有意に低い馬主: **0名**
- 2-3着率が有意に高い馬主: **0名**
- 補正なし（p<0.05）で勝率低+2-3着率高の**両方に該当する馬主: 0名**

もし着順調整があるなら「勝率が低く、かつ2-3着率が高い」馬主が存在するはずだが、そのようなケースはなかった。勝率が低い馬主は4着以下も多く、馬の実力不足と解釈できる。

### Step 4: 連続着順パターン分析

人気上位（1-3番人気）なのに2-3着を連続する馬の頻度を検証した。

| ストリーク長 | 観測数 | 独立試行の期待数 | 比率 |
|------------|--------|----------------|------|
| 2連続以上 | 3,041 | 3,456 | 0.88 |
| 3連続以上 | 944 | 768 | 1.23 |
| 4連続以上 | 317 | 169 | 1.88 |
| 5連続以上 | 113 | 37 | 3.10 |

長いストリークが期待の2-3倍あるが、これは各出走を独立試行と仮定した場合の比較。実際には「善戦マン」体質の馬（実力はあるが勝ちきれない）が存在するため、個体差を考慮すれば自然に説明可能。

上位にリストされた馬主（サンデーレーシング、シルクレーシング、社台レースホース等）はいずれも大規模クラブ法人であり、保有頭数が多いためケース数が多いだけで、率として異常ではない。

### Step 5: オッズ乖離分析

単勝オッズの暗示確率（1/オッズ）と実際の勝率の乖離を馬主別に分析した。

- 昇格リスクあり群は暗示確率に対して残差が-0.056、リスクなし群は-0.045（差=-0.011、p=0.022）
- 統計的に有意だが効果量は**1.1ポイント**とごく小さい
- 64名の馬主がオッズに対して有意に低いパフォーマンスを示すが、これはオッズの「お気に入り-長打バイアス」（人気馬は控除率の影響で期待値が低い）による自然な現象であり、着順調整の証拠とは言えない

## 結論

| 検証項目 | 結果 | 仮説を支持するか |
|---------|------|----------------|
| 昇格リスク馬の勝率低下 | 有意でない（p=0.23） | No |
| 降級廃止後の効果拡大 | 有意でない（p=0.56） | No |
| 馬主別の異常な勝率（N>=50） | 0名検出 | No |
| 馬主別の異常な勝率（N=20-49） | 0名検出 | No |
| 馬主別の異常な2-3着率 | 0名検出 | No |
| 連続2-3着ストリーク | 長いストリーク多め | 個体差で説明可能 |
| オッズ乖離 | 弱い効果あり | バイアスで説明可能 |

11年分・51万出走のデータで多角的に検証した結果、「馬主が意図的に着順を抑えている」ことを示す統計的証拠は得られなかった。

「善戦マン」（人気上位でも2-3着を繰り返す馬）は確かに存在するが、これは馬の勝ちきれない体質として自然に説明でき、特定の馬主に偏った現象ではない。

### 分析の限界

- 統計分析はパターンの検出であり、個別ケースの意図を証明することはできない
- 極めて少数（年に1-2回程度）の着順調整は、統計的なノイズに埋もれて検出不可能
- 走法データ（コーナー通過順位→最終着順の変化）による個別ケース分析は今回実施していない
